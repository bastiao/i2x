i2x

Integrate Anything

Integration-as-a-Service Platform

Integration-as-a-Service Platform

i2x open-source platform brings the power of automated realtime integration to everyone. Quickly create new workspaces integrating data from miscellaneous data sources into any working environment.

Features

Automation
Create custom automated integration pipelines. No more scripting and manually launching tasks, with i2x the platform does the work for you.
More...
Integration
Integrate everything. If there's a way to get the data, i2x can get it for you. From complex SQL queries to light JSON objects, multiple formats can be pushed into your custom i2x workspace.
More...
Real time
No more integration snapshots. Where traditional data integration strategies create a unique version of aggregated data, with i2x the integration is continuous and in (near) real time.
More...
Semantics
Liberate your data. With a Semantic Web-based ETL engine, i2x abstracts data to a semantic environment for integration, analysis and processing.
More...
Rules
An innovative semantics-based rule processing engine allows you to process acquired data before, during and after the final integration tasks.
More...
Templates
Delivery templates to integrate everywhere. From custom SQL queries, to file appends, to web service calls up to sending emails, with i2x integrated data can be published everywhere.
More...
